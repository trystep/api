# Тесты методов get, put, post, delete, path

<h3>Как запустить выполнение тестов в консоли?</h3>
<li>python -m venv env</li>
<li>source env/bin/activate</li>
<li>pip install -U pip</li>
<li>pip install -r requirements.txt</li>
<li>pytest .</li>
